# Методический лонгрид: От НИР к готовой ВКР

Интерактивное методическое руководство по написанию, оформлению и защите выпускной квалификационной работы (ВКР).

## 📖 Описание

Этот проект представляет собой одностраничное веб-приложение (HTML/CSS/JS) с интерактивным аккордеонным меню, содержащее подробные методические рекомендации для студентов по подготовке ВКР.

## ✨ Особенности

- **Адаптивный дизайн** — корректно отображается на всех устройствах
- **Интерактивная навигация** — оглавление с быстрым переходом к разделам
- **Аккордеонная структура** — компактное представление информации
- **Индикатор прогресса** — показывает текущую позицию при чтении
- **Современный UI** — приятный визуальный стиль с градиентами и анимациями

## 📚 Содержание

1. **Структура ВКР** — требования к объёму и разделам работы
2. **Введение** — как правильно оформить вводную часть
3. **Глава 1: Теоретическая часть** — обзор источников и обоснование методов
4. **Глава 2: Эмпирическое исследование** — практическая часть работы
5. **Глава 3: Прикладное решение** — разработка рекомендаций
6. **Выводы и заключение** — завершающая часть
7. **Оформление работы** — технические требования
8. **Список литературы** — правила оформления источников
9. **Приложения** — дополнительные материалы

## 🚀 Использование

Просто откройте файл `index.html` в любом современном браузере:

```bash
# Локально
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```

Или используйте локальный веб-сервер:

```bash
# Python 3
python -m http.server 8000

# Node.js (если установлен http-server)
npx http-server -p 8000
```

Затем откройте в браузере: `http://localhost:8000`
